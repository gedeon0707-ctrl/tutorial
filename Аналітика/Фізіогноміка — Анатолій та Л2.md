# 🔍 Фізіогноміка + астро — Анатолій Буряченко та Л2

## Буряченко Анатолій (☉ Водолій 14°, 03.02.1978)

### Зовнішність
- Овально-прямокутне обличчя з м'якими контурами
- Високий широкий лоб — стратегічне мислення
- Світлі очі, прямий погляд — відкритість з оцінкою
- Підборіддя округле — не агресор, але наполегливий
- Загальний вигляд: "свій" у будь-якому кабінеті

### Астро-збіг
| Риса | Планета | Як проявляється |
|------|---------|-----------------|
| Широкий лоб, стратег | ☉ Водолій | Бачить картину цілком |
| М'яка дипломатія | ♀ Риби | Обходить, не йде в лоб |
| Прямий погляд з оцінкою | ☿ Козеріг | Прораховує, зважує |
| Наполегливий тиск | ♂ Лев | Тисне "по-царськи", не грубо |
| "Своя" людина | ♃ Близнюки | Комунікабельний, має зв'язки скрізь |

### Поведінковий портрет
- Тисне м'яко але постійно ("ну що там?" кожні 2 дні)
- Працює через систему зв'язків: Ференс, Буханевич, Михайло — ніколи напряму до Сугак
- Виглядає безпечним — тому його пускають всюди
- ♆ □ Пн.Вузол Сергія (0°!) — щиро обіцяє те що не завжди контролює

## Л2 — абʼюзер (☉ Водолій 29.8°, 18.02.1997)

### Зовнішність
- Овально-округле обличчя, м'які контури — "місячне" (☽ Рак)
- Спокійний доброзичливий вираз, посмішка — фасад безпечності
- Міцна атлетична статура — фізична сила за м'яким фасадом
- Очі прямі, впевнені, але без глибини — поверхневий контакт
- Округле підборіддя — слабка воля, компенсує фізичною силою
- Загальний вигляд: "ідеальний бойфренд для Instagram"

### Астро-збіг
| Риса | Планета | Як проявляється |
|------|---------|-----------------|
| М'яке обличчя, "безпечний" | ☽ Рак | Зовнішня м'якість, емоційна маніпуляція |
| Посмішка-фасад | ♆ ☍ ♂ | Небезпека за нормальністю |
| Атлетична статура | ♂ Терези | Сила для контролю, не захисту |
| Поверхневий погляд | ☉ Водолій | Дивиться "на тебе", не "в тебе" |
| Фото: спорт, подорожі, SUP | ♂ △ ♃/♅ | Енергійний, авантюрний, зовні активний |

### Поведінковий портрет
- Виглядає як "хороший хлопець" — тому Л1 повертається
- Дає ілюзію дому (♆ ☍ ♂), не справжній дім
- Ініціює дії замість Л1 (його ☽ активує її ♂) — "давай поїдемо"
- Фізичне насильство, переслідування, поліція
- Шарм чергується зі зривами — ♆ тримає 2 роки (до 2028)

### Чому Л1 не йде
- Він виглядає безпечно (☽ Рак = "домашній")
- Його ☽ 16.1° ≈ її ♂ 14.4° — він запускає її дію без зусиль
- ♆ ☍ ♂ = "загадка яку хочеться розгадати"
- Для неї Сергій і Л2 — в одній точці (14-16° Рак), підсвідомість не розрізняє
