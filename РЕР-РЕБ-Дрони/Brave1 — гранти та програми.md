# 🇺🇦 Brave1 — державна платформа оборонних технологій

## Що це
Єдина державна координаційна платформа (кластер) для розвитку оборонних технологій. Об'єднує військових, розробників, інвесторів та уряд.

**Портал:** brave1.gov.ua
**Гранти:** grants.brave1.tech

---

## Рівні фінансування (по TRL)

| Сума | Стадія | Що фінансується |
|------|--------|----------------|
| **до 500 000 грн** | R&D | Тестування ідеї, наукові дослідження, перевірка гіпотез |
| **до 2 000 000 грн** | Лабораторний зразок | Підтвердження працездатності технології |
| **до 4 000 000 грн** | Прототип | Полігонні випробування |
| **до 8 000 000 грн** | Серія | Масштабування, серійне виробництво, бойові тести |

---

## 20 пріоритетних напрямків (2026)

| Категорія | Приклади |
|-----------|---------|
| **РЕБ та РЕР** | Виявлення та подавлення дронів, радарів, зв'язку |
| **БПЛА** | Розвідувальні, ударні, перехоплювачі, "дрони-матки" |
| **Наземні роботи (НРК)** | Логістичні, бойові, мінні платформи |
| **Штучний інтелект** | Автономна навігація, розпізнавання цілей, аналіз РЕР |
| **Компоненти (НОВИНКА)** | Мотори, антени, конектори, камери, плати керування |

---

## "Brave1 Components" — програма для деталей

- **До 8 млн грн** для виробників критичних компонентів
- Мета: **незалежність від Китаю**
- Фокус: мотори, антени, ВЧ-конектори, камери, плати

---

## Процес отримання гранту

```
1. Реєстрація → brave1.gov.ua
2. Експертиза → військові + технічні експерти
3. Статус BRV1 → висока пріоритетність
4. Фінансування → траншами під KPI
5. Кодифікація → допуск до експлуатації + держконтракт
```

---

## Головна вимога 2026

> **30% локалізації** — для участі в держзакупівлях обов'язково мати 30% українських складників у собівартості.

---

## Бізнес-можливість

Ланцюжок для виробництва українських ВЧ-конекторів:
1. **ЧПУ** (TF-CNC, Спецреммаш) — корпус конектора
2. **Гальваніка** (Гальванік-Декор) — золочення контактів
3. **Збірка** — кабель (Одескабель) + конектор + герметизація
4. **Результат** = "український компонент" → грант Brave1 + держконтракти

---

## Релевантність

### Для управління ДПСУ:
- Замовляти через Brave1 спеціалізовані рішення для кордону
- Бути тестувальним полігоном для нових розробок
- Формувати ТЗ → Brave1 знаходить виконавця

### Для бізнесу (окремий трек):
- Виробництво компонентів = ринок з держпідтримкою
- Brave1 Components + 30% локалізація = бар'єр для китайського імпорту
- Грант до 8 млн + масштабування = серійне виробництво
