# 🔌 ВЧ-кабелі та конектори — вузьке місце українського РЕБ

## Ключова проблема
> Модулі РЕБ Україна навчилась збирати. Але **ВЧ-конектори (SMA, N-type, QMA, BNC)** майже не виробляються — масово закуповуються в Китаї з нестабільною якістю.

---

## 1. Виробники кабелів в Україні

| Компанія | Продукція | Примітка |
|----------|-----------|----------|
| **Одескабель** | Коаксіальні RG-213, RG-223, RG-58 (50 Ом) | Головний гігант |
| **Львівкабель** | Кабелі для військового зв'язку, AWG-стандарти | Держзамовлення |
| **Ютекс Україна** | Оптоволоконні кабелі | Критично для "дронів на дроті" — попит зріс у десятки разів |

---

## 2. Три критичних бар'єри

### Бар'єр 1: Дефіцит конекторів
- SMA, N-type, QMA, BNC — в Україні **майже не виробляються**
- Закуповуються в Китаї (AliExpress/Alibaba) або Тайвані
- Якість "плаває" → великі втрати сигналу → РЕБ неефективний

### Бар'єр 2: Кабельні збірки
- РЕБ потребує кабель + конектори + герметизація
- Попит 2026: **мільйони одиниць** (план 7 млн дронів)
- Не вистачає автоматизованих ліній обтискання

### Бар'єр 3: Спецматеріали
- Тефлон/фторопласт для діелектриків — імпортний
- Якісна мідь — імпортна
- Ціна українського кабелю на **20-30% вища** за китайський

---

## 3. Чому важко робити конектори в Україні?

| Процес | Складність | Вимога |
|--------|-----------|--------|
| **ЧПУ-обробка** | Точність до 0.005 мм | Похибка → "стояча хвиля" → згорає підсилювач |
| **Діелектрики** | Тефлонова вставка | Специфічне обладнання для дрібних деталей |
| **Гальваніка** | Золочення центрального контакту (кілька мікрон) | Без золота → окислення → зростання опору |

## Релевантність для управління ДПСУ

Управління як **замовник** має розуміти:
1. Якість конекторів = ефективність РЕБ
2. Вимагати українські збірки (локалізація 30%)
3. Тестувати якість поставок (RigExpert — вимірювальні прилади)
4. Працювати з Brave1 для розвитку компонентної бази
