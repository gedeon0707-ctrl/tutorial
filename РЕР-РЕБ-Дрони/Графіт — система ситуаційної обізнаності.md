# Графіт — система ситуаційної обізнаності

## Що це
Цифрова система моніторингу повітряного простору та виявлення загроз у режимі реального часу. Розроблена українськими військовими IT-фахівцями для ЗСУ та Сил оборони.

## Ключові функції
- **Моніторинг загроз** — миттєве сповіщення про ворожі розвідувальні/ударні БПЛА та запуск КАБ (керованих авіаційних бомб)
- **Оперативна передача** — без затримок передає інформацію про ворожі цілі вищому командуванню та суміжним підрозділам
- **Інтеграція з Армія+** — доступ через застосунок Армія+ за поданням командира
- **Графіт Лайт** — полегшена версія / рівень доступу для базового моніторингу

## Місце в екосистемі

```
Ланцюжок протидії БПЛА:

  ГРАФІТ (виявлення) → РЕБ (подавлення) → DELTA (координація)
       ↑                    ↑                    ↑
     РЕР-підрозділ     РЕБ-підрозділ      Штаб / Командування
```

| Система | Роль | Тип |
|---------|------|-----|
| **Графіт** | Виявлення повітряних загроз (БПЛА, КАБ) | РЕР / ситуаційна обізнаність |
| **DELTA** | Загальна бойова картина, координація | Mission Control |
| **Дозор AI (VARTA)** | AI-аналіз загроз | Розвідка |
| **РЕБ-системи** (Kvertus, INFOZAHYST) | Подавлення / нейтралізація | Радіоелектронна боротьба |

## Зв'язок з РЕР
Графіт = інструмент **радіоелектронної розвідки** в частині повітряного моніторингу. Підрозділ РЕР використовує дані Графіту для:
1. Виявлення типу БПЛА (розвідувальний / ударний)
2. Визначення напрямку та швидкості
3. Передачі цілевказівки для РЕБ
4. Документування повітряної обстановки

## Доступ
- Через застосунок **Армія+**
- Потребує подання командира підрозділу
- Навчальний курс інтегрований в Армія+
