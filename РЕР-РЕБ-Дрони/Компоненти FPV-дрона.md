# 🎮 Компоненти FPV-дрона

## Основні елементи
| Компонент | Функція | РЕБ-вразливість |
|-----------|---------|-----------------|
| **Польотний контролер** | Мозок — утримує дрон у повітрі | Ні |
| **Ресівер управління** | Приймає команди від пілота | ✅ 2.4 ГГц / 900 МГц |
| **Відеотрансмітер (VTX)** | Передає відео пілоту | ✅ 5.8 ГГц / 1.2 ГГц |
| **Камера** | Зображення для пілота | Ні |
| **ESC (регулятор обертів)** | Контролює мотори | Ні |
| **Мотори** | Тяга | Ні |
| **Антена управління** | Діпольна, приймає сигнал | ✅ Ціль для РЕБ |
| **Антена відео** | Передає FPV-відео | ✅ Ціль для РЕБ |

## Ключові частоти
| Частота | Що передає | РЕБ подавляє |
|---------|-----------|-------------|
| 2.4 ГГц | Управління | ✅ |
| 5.8 ГГц | FPV відео | ✅ |
| 900 МГц | Далекобійне управління | ✅ |
| 1.2 ГГц | Аналогове відео | ✅ |
| GPS L1 (1575 МГц) | Навігація | ✅ |

## Захист від РЕБ
- **Волоконний FPV** — управління по оптоволокну (імунний до РЕБ)
- **AI-донаведення** — дрон летить автономно після втрати зв'язку
- **Frequency hopping** — швидка зміна частот
- **Mesh-мережі** — дрони передають сигнал один одному
