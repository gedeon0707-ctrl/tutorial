# 📡 Виробництво антен для РЕБ та РЕР в Україні (2026)

## Ключовий факт
> Раніше антени купували в Китаї. Зараз більшість топових компаній мають **власне замкнене виробництво** або працюють з українськими виробниками компонентів. **300+ компаній** задіяні у виробництві РЕБ та компонентів.

---

## Ключові виробники антенних систем

### Warmax Technologies
- Антенно-модульні комплекси для РЕБ та БПЛА
- **Кодифікація НАТО** ✅
- Інтеграція з оптоволоконними системами керування

### Kvertus (Квертус)
- З 2025 — **повністю власне виробництво антен** (незалежність від імпорту)
- Спрямовані антени для рушниць
- Всенаправлені купольні антени для стаціонарних комплексів

### Unwave
- **300+ працівників**
- Антени, підсилювачі, генератори
- Компоненти стандартів НАТО

### Щит Землі (ЩЗ)
- Квадрофілярні, колінеарні, турнікетні антени
- "Гнучкі" рішення — антена налаштовується під конкретний діапазон замовника

### Spectron
- Схрещені турнікетні антени (тип Моксон)
- Кругова поляризація
- Висока ефективність проти FPV-дронів

## Типи антен українського виробництва

| Тип | Застосування | Особливість |
|-----|-------------|-------------|
| **Логоперіодичні** (спрямовані) | РЕР — визначення координат; антидронові рушниці | Точне спрямування |
| **Квадрофілярні та турнікетні** | Купольний РЕБ | Рівномірне покриття, кругова поляризація |
| **Колінеарні** (штирьові) | Захист бронетехніки | Високе підсилення, кругова діаграма |
| **Фазовані антенні решітки (ФАР)** | Smart-РЕБ — адаптивне керування | Точний удар завадою без засліплення своїх |

### ФАР — найскладніший сегмент
Україна **почала власне виробництво** ФАР для адаптивного керування променем. Smart-РЕБ дозволяє "бити" завадою точно в ціль, не засліплюючи свої засоби зв'язку. Це вирішує проблему [[Контроль за застосуванням РЕБ|контролю РЕБ]].

## Brave1 — грантова підтримка
- Програма грантів **до 8 млн грн** для виробників компонентів (зокрема антен)
- Фінансування: високоточні верстати для друкованих плат антен
- Вимірювальне обладнання: векторні аналізатори ланцюгів

## Релевантність для управління ДПСУ

Нове управління може:
1. **Замовляти** антени під специфіку кордону (діапазони контрабандистських дронів)
2. **Тестувати** ФАР/Smart-РЕБ на прикордонних об'єктах
3. **Працювати з Brave1** — гранти на розробку спеціалізованих рішень
4. **Формувати ТЗ** — які частоти, яка поляризація, який радіус потрібен ДПСУ
