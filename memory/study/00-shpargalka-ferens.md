## 🏛️ 1. ПОЛОЖЕННЯ ПРО МРУ (Наказ №1707/5)

**Що таке МРУ:**
- Територіальний орган Мін'юсту, юридична особа публічного права
- Утворюється/ліквідується **КМУ** за поданням Міністра
- Начальник призначається **Міністром юстиції**
- Перший заступник — призначається Міністром **за поданням начальника**

**Напрямки МРУ:** РАЦС, держреєстрація (ЮО/ФОП/ГФ + речові права), нотаріат, виконавча служба, легалізація ГО, правова просвіта, БПД

**Перший заступник:**
- Виконує обов'язки начальника у разі відсутності
- Координує закріплені напрямки
- Головує в комісії з розгляду скарг
- Несе персональну відповідальність за свої напрямки

## ⚖️ 2. НОТАРІАТ (ЗУ №3425-XII + Наказ №296/5)

### Ключові цифри
| Параметр | Значення |
|----------|----------|
| Стаж для нотаріуса | **6 років** у сфері права, з них **3 роки** помічником |
| Страхування відповідальності | мін. **1000 мінімальних зарплат** |
| Планові перевірки | не частіше **1 раз на 3 роки** |
| Позапланові перевірки | за скаргою, дорученням Мін'юсту, рішенням суду |

### Хто вчиняє нотаріальні дії
- Нотаріуси (державні та приватні)
- Посадові особи ОМС (обмежене коло — де немає нотаріуса)
- Консульські установи (за кордоном)

### Контроль МРУ за нотаріусами
- Перевірки організації роботи (планові/позапланові)
- Ведення реєстру приватних нотаріусів округу
- Подання до ВККН про анулювання свідоцтва
- Зупинення/припинення нотаріальної діяльності
- Розгляд скарг на організацію роботи

### Нотаріальна таємниця (ст. 8)
- Довідки видаються: учасникам, суду, прокуратурі, слідчим
- Вилучення документів нотаріуса — **лише за рішенням суду**

### Відмова нотаріуса (ст. 49)
- Оформляється **письмово** із зазначенням причин
- Оскаржується **до суду** (ст. 50)

### 14 видів нотаріальних дій (ст. 34)
Правочини, спадщина, довіреності, копії, підписи, факти, виконавчі написи, депозит, протести, зберігання, реєстрація прав, реєстрація ЮО/ФОП

## 📝 3. РЕЄСТРАЦІЯ ЮО, ФОП та ГФ (ЗУ №755-IV)

### Ключові строки
| Дія | Строк |
|-----|-------|
| Реєстрація створення ЮО | **24 години** |
| Зупинення розгляду (усунення недоліків) | до **15 робочих днів** |
| Камеральна перевірка | **30 робочих днів** |
| Розгляд скарги | **45 календарних днів** (до 90) |

### Суб'єкти реєстрації
- Виконавчі органи рад
- Нотаріуси
- Акредитовані суб'єкти

### Контроль МРУ
- **Моніторинг** реєстраційних дій в ЄДР (автоматизований)
- **Камеральна перевірка** документів
- **Розгляд скарг** на реєстраторів
- **Тимчасове блокування** доступу реєстратора (до 2 міс.)
- **Анулювання** доступу
- **Складання протоколів** про адмінправопорушення

### Підстави для відмови в реєстрації (ст. 18)
Неповний пакет, невідповідність документів, неуповноважена особа, невідповідність установчих документів, неправильне найменування, несплата збору

## 🏠 4. РЕЄСТРАЦІЯ РЕЧОВИХ ПРАВ (ЗУ №1952-IV)

### Ключові строки
| Дія | Строк |
|-----|-------|
| Загальний строк реєстрації | **5 робочих днів** |
| Скорочений (за доплату) | 2 дні / 1 день / **2 години** |
| Нотаріус | **негайно** (одночасно з нотаріальною дією) |
| Зупинення (усунення недоліків) | до **15 робочих днів** |
| Блокування доступу реєстратора | до **3 місяців** |

### Що реєструється
- **Права:** власність, оренда (≥3 роки), сервітут, емфітевзис, суперфіцій, іпотека, довірча власність
- **Обтяження:** заборона відчуження, арешт, іпотека, податкова застава

### Адміністративний збір (ст. 34)
| Дія | Розмір |
|-----|--------|
| Реєстрація права власності | 0,1 прожиткового мінімуму |
| Інші речові права | 0,05 прожиткового мінімуму |
| Обтяження | 0,04 прожиткового мінімуму |

## 📨 5. ПОРЯДОК РОЗГЛЯДУ СКАРГ (ПКМ №1128)

### Строки
| Дія | Строк |
|-----|-------|
| Подання скарги | **60 календарних днів** з моменту, коли дізнався |
| Розгляд скарги | **45 календарних днів** |
| Продовження розгляду | до **60 днів** (потрібні додаткові документи) |
| Повідомлення про результат | **3 робочих дні** після рішення |

### Комісія з розгляду скарг при МРУ
- **Голова** — заступник начальника (= ТИ)
- Непарна кількість, **не менше 5** членів
- Кворум — **2/3** складу
- Рішення — **більшістю** голосів
- Протокол обов'язковий
- Скаржник має право бути присутнім

### Рішення комісії (3 варіанти)
1. ✅ **Задоволити:** скасувати рішення/дію, зобов'язати вчинити дію
2. ❌ **Відмовити** у задоволенні
3. ⏸️ **Залишити без розгляду:** пропущено строк, є рішення суду, вже розглянуто

### Оскарження рішення комісії МРУ
- Адміністративне → до **Мін'юсту**
- Судове → до **адміністративного суду**

## 👶 6. РАЦС (ЗУ №2398-VI + Накази №52/5, №96/5)

### Строки реєстрації
| Акт | Строк подання |
|-----|---------------|
| Народження | **30 днів** (1 місяць) |
| Смерть | **3 дні** |
| Шлюб (очікування) | **1 місяць** після заяви |
| Розірвання шлюбу (РАЦС) | **1 місяць** |
| Зміна імені (розгляд) | **3 місяці** (до 6) |

### 7 видів актів цивільного стану
1. Народження
2. Смерть
3. Шлюб
4. Розірвання шлюбу
5. Зміна імені
6. Усиновлення
7. Встановлення батьківства

### Органи РАЦС
- Відділи РАЦС (підрозділи МРУ) — основні
- Виконкоми сільських/селищних рад — обмежений обсяг
- Консульські установи — за кордоном

### Шлюбний вік
- **18 років** (за рішенням суду — з **16**)

### Підстави відмови в зміні імені (ст. 23-25)
- Розшук, непогашена судимість, заборона суду

### ДРАЦС (Державний реєстр АЦС)
- Адміністратор — **Мін'юст**
- Реєстратори — працівники відділів РАЦС

## 📋 7. ЗМІНИ ДО АКТОВИХ ЗАПИСІВ (Наказ №96/5)

### Строки
| Дія | Строк |
|-----|-------|
| Розгляд заяви | **3 місяці** |
| Продовження | до **6 місяців** |
| Оскарження відмови | до **суду** (окреме провадження, ст. 307 ЦПК) |

### 5 видів змін
1. **Виправлення** — усунення помилок/описок
2. **Доповнення** — пропущені відомості
3. **Зміна** — зміна змісту запису
4. **Анулювання** — лише за **рішенням суду**
5. **Поновлення** — відновлення втраченого запису

### Підстави для змін
- Рішення суду
- Рішення відділу РАЦС
- Заява заінтересованої особи
- Інші документи за законом
