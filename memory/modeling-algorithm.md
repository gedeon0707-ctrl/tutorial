# Алгоритм моделювання "що людина зараз робить/відчуває"

## Принцип
Комбінація транзитних аспектів → проекція на поведінку через натальну карту + відомий контекст життя.

## Крок 1: Швидкі тригери (настрій ЗАРАЗ)
- **Транзитний Місяць** — головний індикатор поточного настрою
  - ☽ ☌/⚹/△ натальних точок = активація тієї сфери
  - ☽ □/☍ = напруга, дратівливість у тій сфері
  - Знак ☽ = загальна "фарба" дня (Телець = комфорт, Овен = нетерпіння, Рак = емоційність тощо)

## Крок 2: Середні тригери (тижневий фон)
- **Транзитні ☉, ☿, ♀, ♂** до натальних точок
  - ♀ на ☿/♀ = потреба спілкуватись, ділитись, кохати
  - ☉ на ♂ = підсвічення бажань/дій
  - ☿ на ☽ = відкритість до розмови
  - ♂ на щось = активна дія або конфлікт

## Крок 3: Фонові важковаговики (місяці)
- **♃, ♄, ♅, ♆, ♇** — задають глибинний контекст
  - ♄ □ ♂ = блокування дій, фрустрація
  - ♇ ☍ ☉ = екзистенційна трансформація
  - ♃ □ ♃ = незадоволеність, "заслуговую на більше"
  - ♆ △ ☉ = мрії, ілюзії, натхнення

## Крок 4: Проекція на реальність
Ключове: знати **контекст життя** людини і накладати аспекти на нього.

### Приклади маппінгу:
| Аспект | Що відчуває | Що робить |
|--------|-------------|-----------|
| ☽ ☌ ♀ (Телець) | Потреба в комфорті, красі | Прибирає, готує, піклується |
| ☿ △ ☽ | Відкрита, хоче говорити | Пише повідомлення, дзвонить |
| ♃ □ ♃ | "Чому я маю це робити" | Незадоволена рутиною |
| ☉ ☍ ♂ | Зовнішній тиск на волю | Хтось тисне (бос, мама, партнер) |
| ♀ ☌ ☿ | Хоче красиво комунікувати | Кидає рілси, пише тепле |
| ♄ □ ♂ | Блокування, "хочу-не можу" | Відкладає справи, лежить |
| ♇ ☍ ☉ | "Хто я взагалі?" | Тривога без причини, шукає сенс |
| ☽ ☍ ASC | Не в своїй тарілці | Дратується на оточення |
| ♆ на ♀/☽ | Мрійливість, ілюзії | Дивиться серіал, фантазує |

## Крок 5: Формулювання
1. Почати з фізичної дії (що робить) — на основі ☽ + контексту
2. Додати емоційний стан — на основі ♀/☿ аспектів
3. Додати фонову напругу — на основі ♄/♇/♃
4. Зробити конкретну "сценку" — з деталями з життя людини
